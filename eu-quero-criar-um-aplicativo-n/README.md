# Professor Central

Primeira base funcional de um aplicativo web para organizar a rotina de um professor.

## O que esta versao mostra

- painel inicial com visao geral
- cadastro de turmas
- cadastro de alunos por turma
- planos de aula
- pendencias
- documentos
- agenda
- salvamento local no navegador

## Como abrir

1. Abra o arquivo `index.html` no navegador.
2. O aplicativo funciona localmente, sem instalar nada.
3. Os cadastros de turmas e alunos ficam salvos no navegador pelo `localStorage`.

## O que vem depois

As proximas etapas naturais para transformar isso em um sistema real sao:

1. adicionar notas e presenca
2. permitir editar e excluir turmas
3. permitir upload de documentos
4. criar login para acessar de qualquer lugar
5. conectar banco de dados online
6. publicar online
